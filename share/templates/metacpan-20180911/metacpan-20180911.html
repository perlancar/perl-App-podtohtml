NAME

Perl::Examples::Module::One - Example module one

VERSION

This document describes version 0.092 of Perl::Examples::Module::One (from Perl distribution Perl-Examples), released on 2018-09-11.

SYNOPSIS

use Perl::Examples::Module::One qw(fisbus);
 
print fisbus(3); # 2
print fisbus(4); # 4
print fisbus(5); # 0
print fisbus(6); # 7

DESCRIPTION

This is an example module with the following features:

  • $VERSION

  • Some code

  • Synopsis

  • Documentation on function

FUNCTION

fisbus

Usage:

fisbus($num) => $new_num

Accept a number then return another number. If input number is even, return number plus one. If input number is divisible by five, return number minus five. Otherwise, return input number minus one.
